# Auralis


Auralis (pronounced “aw-RAL-iss”) is a Windows desktop audio automation application built with C# (.NET 8, WPF) using NAudio (WASAPI).

## Features
- Monitors a primary audio source (e.g. Chrome / YouTube)
- Smoothly fades a secondary source (e.g. Spotify)
- Restores audio after silence
- Per-application session volume control
- Primary audio is never modified

## Tech Stack
- C# / .NET 8
- WPF
- NAudio (WASAPI)
